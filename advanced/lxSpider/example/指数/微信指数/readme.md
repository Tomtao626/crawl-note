#### 说明
1. main.js 里面写的是 查询微信公众号的 demo 和 查询微信搜索指数的 demo
2. js 的执行流程类似于 python 的执行流程，开发时不要有太大的压力
3. 可以安装 visual studio code 工具，然后安装 auto.js 插件。这样就可以在电脑上写代码，然后在手机 / 模拟器上运行。
4. 调试的机器需要使用 android 7.0 以上的手机或者模拟器。
